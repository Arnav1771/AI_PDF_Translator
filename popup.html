How to Use:

  1. Get a Gemini API Key from Google AI Studio or Google Cloud Platform (ensure it's enabled for the Gemini API).
  2. Enter the API Key above and select your desired target language.
  3. Click "Save Settings".
  4. Open a PDF file in your Chrome browser.
  5. Select the text you want to translate within the PDF.
  6. Right-click on the selected text.
  7. Choose "Translate with TranslateExt" from the context menu.
  8. The translation will appear as a browser notification (and in the "Last Translation" box above next time you open this popup).
